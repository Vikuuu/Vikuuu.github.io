Inverted Index implementation in Go

A toy implementation of **Inverted Index** in go, that powers most if not all the **Information Retrieval** for
_web engine_, _Elasticsearch_, and more.

## Example

Doc 1

I did enact Julius Caesar: I was killed i' the Capiot; Brutus killed me.

Doc 2

So let it be with Caesar. The noble Brutus hath told you Caesar was ambitious.
